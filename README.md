## MVC Entity Framework Assignment

This project is an ASP.NET MVC web application that calculates a car insurance quote based on user input. The application uses business logic to calculate the monthly insurance quote. It also includes an Admin page where staff can view issued quotes with the user's first name, last name, email address, and quote amount.

## Final Assignment Code First
